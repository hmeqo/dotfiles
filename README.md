# DotFiles By Colin Iliad

## Install

### Softlink

Use [xdm](https://github.com/XXiaoA/xdm) to manage dotfiles.

```bash
./install.sh
```

### Fonts

- Fira Code (Text)
- FiraCode Nerd Font (Nerd Font)
- Noto Color Emoji (Emoji)
- Symbols Nerd Font Mono (Icon)
- Noto Sans CJK SC (Chinese)
