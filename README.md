# Personal dotfiles

## Install

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Usage

```bash
# source .venv/bin/activate
# venv
scstui
```

### Fonts recommended

- Fira Code (Text)
- FiraCode Nerd Font (Nerd Font)
- Noto Color Emoji (Emoji)
- Symbols Nerd Font Mono (Icon)
- Noto Sans CJK SC (Chinese)
