# Personal dotfiles

## Install

### Use venv

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Use pdm or other package manager

```bash
pdm install
```

## Usage

```bash
# venv
scstui
# pdm
pdm run scstui
```

### Fonts recommended

- Fira Code (Text)
- FiraCode Nerd Font (Nerd Font)
- Noto Color Emoji (Emoji)
- Symbols Nerd Font Mono (Icon)
- Noto Sans CJK SC (Chinese)
