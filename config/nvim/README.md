## Directory structure

- lua
  - community
    Custom Astronvim community pack
    - astrodefault
  - plugins
    Custom plugins
    - astrodefault
    - overrides
  - presets
    Like Astronvim community, no loaded by default
  - polish
  - user
    - community
      User custom Astronvim community pack or presets
    - plugins
      User custom plugins
    - polish
- snippets
  - user
- user-config
