## Directory structure

- lua
  - core
    Tool and code
  - default
    Astronvim default config
  - plugins
    Custom plugins
  - presets
    Like Astronvim community, no loaded by default
  - community
    Custom Astronvim community pack
  - polish
  - user
    - plugins
      User custom plugins
    - community
      User custom Astronvim community pack or presets
    - polish
- snippets
  - user
- user-config
